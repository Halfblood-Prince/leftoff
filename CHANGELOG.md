# Changelog

## 1.0.0 - 2026-07-06

- Added local store initialization, validation, backup-before-repair, and secret rejection.
- Added explicit capture for tasks, ideas, decisions, problems, solutions, open loops, releases, and friction events.
- Added read-only Git inspection, project state updates, and resume packets.
- Added explainable `/now` prioritisation.
- Added decision recall, solved-problem search, weekly review, friction detection, cleanup advisor, and optional GitHub CLI metadata caching.
- Added AI-agent compatibility docs, installer dry-run mode, export/import, and guarded delete-data flow.
- Added officially tested Claude Code, Codex, and GitHub Agent Skills CLI paths, plus compatible standard `SKILL.md` guidance and experimental adapter notes for additional hosts.
- Added macOS CI coverage, CodeQL scanning, Go fuzzing, and annotated-tag release packaging for `gh skill`-style installs.

The v1.0 core workflow requires no network access and performs no destructive action by default.
