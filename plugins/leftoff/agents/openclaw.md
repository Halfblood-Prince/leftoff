# OpenClaw Adapter

Status: Supported
Installer aliases: `openclaw`
Suggested target: `~/.openclaw/skills/leftoff/`

Use this adapter for OpenClaw-based agents that support reusable local instructions or skill folders.

If OpenClaw uses a workspace-specific location, copy or symlink the repository there and load `skills/leftoff/SKILL.md` first. Keep shared behavior in `skills/leftoff/SKILL.md` and detailed reference material in `skills/leftoff/references/`.
