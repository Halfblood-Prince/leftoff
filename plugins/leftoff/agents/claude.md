# Claude Code Adapter

Status: Supported
Installer aliases: `claude-code`, `claude`
Suggested target: `~/.claude/skills/leftoff/`

Install or symlink the repository into the Claude Code skill directory, then let Claude Code load `skills/leftoff/SKILL.md` as the canonical instruction file.

Use the same CLI commands documented in `skills/leftoff/SKILL.md`. Keep Claude-specific notes here and shared behavior in `skills/leftoff/SKILL.md` or `skills/leftoff/references/`.
