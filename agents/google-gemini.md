# Gemini CLI / Antigravity Adapter

Status: Supported
Installer aliases: `gemini-cli-antigravity`, `gemini`, `google-gemini`, `antigravity`
Suggested target: `~/.gemini/skills/leftoff/`

Use this adapter for Gemini CLI, Antigravity, and Gemini-based assistants that support reusable instructions, extensions, or local skill folders.

If the host uses a different custom-instructions location, place or link this repository there and make `skills/leftoff/SKILL.md` the entry point. Keep the local-first privacy rules unchanged.
